# sound-good

SoundCloud clone app

## Install

Systems required:
- Docker Engine: v26.0.0

Run:
1. Run docker-compose
   `` docker-compose up -d ``
2. Check heath ``http://localhost:8080/sound-good/api/actuator/health``
3. Access the link ``http://localhost:3000``
